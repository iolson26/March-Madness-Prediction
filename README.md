# March-Madness-Prediction
Math modeling project: March madness Predictions using random forest and XGBOOST.

All of our data can be found in the Spreadsheets folder, its mostly filled with game data that can be ignored but the final training dataset is titled Compiled DATA REAL.csv

The code used to API request historical game data is in the retrieve_games_code folder.

Graphs folder contianes all of the graphes generated from our project. 

The final random forest code is in the final_model jupyter notebook. Im afraid that there is a lot of old stuff and random stuff in there, it also isn't marked up super well. Sorry. 

The final xgboost model is the xgboost_model jupyter notebook. Again it isn't marked up super well.
